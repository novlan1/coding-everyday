# 发布npm包

  1. 注册一个npm账号
  2. 开发一个npm包，必须npm init
  3. npm adduser
  4. npm publish

# 项目中应用npm包

npm install xxxx

# 更新npm包

如果之后修改过此包，需要修改package.json中的版本号字段version，使其大于当前版本，然后npm publish即可。

如果未更改版本号，会报错。

修改版本号后则成功:

npm publish

# 删除npm包

npm unpublish
npm unpublish --force
