权限分级系统：系统、S、A、B、C、D...

- 每下一级，node_id就增加@+数字
- 最大一级是N0，也就是系统管理员
- 同一级表示方式举例：N0@1，N0@2，N0@3...
- 比如，N0@22@0@0@0@0就是D级


```ts
/**
 *  N0node_0_admin => 超级
    N0@38node_1_admin => S
    N0@38@0node_2_admin => A
    N0@38@0@0node_3_admin => B 
 */
function getNodeLevel(name = '') {
  const len = name.split('@').length;
  if (len === 1) {
    return '超级';
  }
  if (len === 2) {
    return 'S';
  }
  return String.fromCharCode(len + 62);
}
```
