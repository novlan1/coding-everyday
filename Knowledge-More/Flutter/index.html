# 1. 主流框架对比

  1. cordova：混合式开发框架（Hybrid app）
  2. RN（React Native）：原生app，以view为基础嵌入
  3. Flutter：在渲染技术上，选择自己实现（GDI），有更好的可控性

# 2. Flutter特点

flutter 120fps超高性能 fps：每秒传输帧数，每秒画面数 第三方组件：

https://github.com/Solido/awesome-flutter法国人总结的好用的组件。

# 3. flutter安装

android Studio 安装完成面板 => Configure => plugins => 搜索flutter并安装 插件安装完成后，会在Android Studio的第二项多出来Start a new Flutter project

再次运行flutter doctor会提示让你输入 flutter doctor --android-licenses 同意一堆协议

Android Studio => Tools => AVD manager
